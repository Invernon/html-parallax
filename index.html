Hola
Parallax

!

¿Cómo hacerlo?

Primero, debemos definir nuestro contendor que tendrá la imagen más lejana.

Luego le colocaremos el siguiente estilo

width: 100%;
height: 100vh;
background-attachment: fixed;
perspective: 1px;
transform: translateZ(-1px);
overflow: hidden;

Posteriormente incluimos la propiedad de background-size: cover

Quedando una clase con los siguientes estilos

section{
width: 100%;
height: 100vh;
perspective: 1px;
transform: translateZ(-1px);
overflow: hidden;
background-attachment: fixed;
background-size: cover;
background-position: center; }

¿Bastante sencillo no?

Esta es tan solo una de las muchas formas de realizar este efecto. Es importante recordar lo siguiente:

  • La seccion que queremos colocarle el parallax debemos asignarle la imagen como un fondo
  • La seccion que queremos colocarle el parallax debemos asignarle:
    background-attachment: fixed;
  • EXTRA Si quieres ir colocando capas, para añadir más efectos, puedes hacerlo con position:absolute; transform: translateZ(0.1px);
    tal como hice con la persona.
